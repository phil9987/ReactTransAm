# ReactTransAm
A clone of the Spectrum game TransAm written in ReactJS

## Build and run it
0. make sure npm is installed (i.e. on Linux `sudo apt install npm`)
1. `npm install`
2. `npm start`
3. browser window will open, navigate with arrow keys
